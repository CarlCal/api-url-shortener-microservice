URL Shortener microservice

API Basejump: URL Shortener

Description:

By passing a website as a parameter after /new/ it will be added to the database, simultaneously a six character code will be generated and linked to the site. If you then type the code provided to you in the address bar from the homepage it will redirect you to the original site. If the site is not legit or the code does not exist in the database, you will recive an error.


Generate:

https://carlcal-url-shortener-ms.herokuapp.com/new/http://google.com

Output:

{ "original_url": "http://google.com", "short_url": "https://carlcal-url-shortener-ms.herokuapp.com/9kSDUP" }

Use:

https://carlcal-url-shortener-ms.herokuapp.com/9kSDUP -> google.com
